How does AGRI PLUS work ?

This project will monitor the humidity of the soil, relative humidity, and temperature

The sensors will collect data in real- time and store them in the database. Meanwhile, the application will access the database and prepare graphs. Once the user asks for the current situation or the graphs, the application will display them. Otherwise, the application will analyze the collected data and send an alert when it is necessary.

Main technologies used in AGRI PLUS

AGRI PLUS is carried out with the latest technologies present in the market

Raspberry PI

Is a micro pc. Representing the brain of our product. Making the connnection between the real world and the user possible

Sensors

Our device is composed of many sensors such as DHT11 Sensors, water level sensor and Photoresistor that collect data

Android

An adroid application is associated to the device. Connecting the user to the device so he can get the warnings and the data.

Firebase

Real time database. Stores the data collected and transform it to the application.
